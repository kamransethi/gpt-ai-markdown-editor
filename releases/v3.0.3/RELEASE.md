# Release Notes — v3.0.3

Date: 2026-04-20

## Highlights

- Added a table-aware bullet toolbar toggle for selected table content.
- Disabled automatic `- ` bullet conversion inside table cells to prevent unintended list creation.
- Improved table bullet behavior so the toolbar can both apply and remove bullets consistently.

## Notes

- The new table bullet toggle works on selected table text and maintains predictable formatting.
- Table cell editing is now more stable after the TipTap and markdown pipeline updates.
